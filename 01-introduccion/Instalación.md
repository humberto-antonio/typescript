### Instalación de manera global typescript

```
npm install -g typescript
```

Verificar versión de typescript
```
tsc --versión
```

### Generación de archivo TSConfig.json
```
tsc --init
```

### Modo watch
```
tsc -w
```
o
```
tsc --watch
```
